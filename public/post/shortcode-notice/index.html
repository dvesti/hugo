Shortcodes Notice Preview

March 3, 2018

normal use

Note example:

{{% notice note %}}
A notice disclaimer
{{% /notice %}}

Result: {{% notice note %}} A notice disclaimer {{% /notice %}}

You could custom title :

{{% notice note 笔记 %}}
A notice disclaimer
{{% /notice %}}

{{% notice note 笔记 %}} A notice disclaimer {{% /notice %}}

tip

{{% notice tip %}}
A tip disclaimer
{{% /tip %}}

{{/* notice tip */}} A tip disclaimer {{< /notice >}}

info

{{% notice info %}}
A info disclaimer
{{% /notice %}}

{{% notice info %}} An information disclaimer {{% /notice %}}

warning

{{% notice warning %}}
A warning disclaimer
{{% /notice %}}

{{% notice warning %}} An warning disclaimer {{% /notice %}}
